# Datacamp Data Literacy Professional

The Datacamp skill track for a [Data Literacy Professional](https://app.datacamp.com/learn/skill-tracks/data-literacy-professional) focus on an understanding of data to data storytelling. The overall aim is to gain valuable Data Literacy Skills to tackle the toughest data challenges. The skill track consist of the following courses:

1. [Introduction to Data](/01-Introduction-to-Data.md)
2. [Communicating Data Insights](/02-Communicating-Data-Insights.md)
3. [Introduction to Data Literacy](/03-Introduction-to-Data-Literacy.md)
4. [Introduction to Statistics](/04-Introduction-to-Statistics.md)
5. [Introduction to Data Culture](/05-Introduction-to-Data-Culture.md)
6. [Forming Analytical Questions](/06-Forming-Analytical-Questions.md)
7. [Data Storytelling Concepts](/07-Data-Storytelling-Concepts.md)
8. [Data Literacy Case Study: Remote Working Analysis](/08-Data-Literacy-Case-Study.md)
